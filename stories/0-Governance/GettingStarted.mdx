# Getting Started

Get up and running with the Elevate Design System in 5 minutes.

---

## Quick Install

### Option 1: Git Repository (Recommended)

```bash
npm install git+https://gitlab-paris.sqli.com/SQLI/elevate-design-system.git
```

```css
/* In your main CSS file */
@import '@sqli/elevate-design-system/dist/sqli-design-system.css';
@import '@sqli/elevate-design-system/dist/fonts.css';
```

```html
<!-- Apply theme to your HTML -->
<html data-theme="sqli-light">
```

---

## Your First Component

Once installed, import React components from the library:

```tsx
import { Button, Card, CardBody, CardTitle, Alert } from '@sqli/elevate-design-system';
import '@sqli/elevate-design-system/css';

function App() {
  return (
    <div data-theme="sqli-light">
      {/* Buttons */}
      <Button variant="primary">Primary</Button>
      <Button variant="secondary">Secondary</Button>

      {/* Card */}
      <Card>
        <CardBody>
          <CardTitle>Hello World</CardTitle>
          <p>Welcome to the Elevate Design System!</p>
        </CardBody>
      </Card>

      {/* Alert */}
      <Alert variant="info">You're all set!</Alert>
    </div>
  );
}
```

CSS-only usage is also supported with HTML classes:

```html
<button class="btn btn-primary">Primary</button>
<div class="card bg-base-100 shadow-xl">
  <div class="card-body">
    <h2 class="card-title">Hello World</h2>
  </div>
</div>
```

### Using ElevateTheme Provider (React)

```tsx
import { ElevateTheme, useTheme } from '@sqli/elevate-design-system';

function App() {
  return (
    <ElevateTheme defaultTheme="sqli-light">
      <MyApp />
    </ElevateTheme>
  );
}

function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  return (
    <button onClick={() => setTheme(theme === 'sqli-light' ? 'sqli-dark' : 'sqli-light')}>
      Toggle Theme
    </button>
  );
}
```

### Using data-theme Attribute (HTML)

```javascript
// Toggle theme
document.documentElement.setAttribute('data-theme', 'sqli-dark');
```
