# Getting Started

Get up and running with the Elevate Design System in 5 minutes.

---

## Quick Install

### Option 1: Git Repository (Recommended)

```bash
npm install git+https://gitlab-paris.sqli.com/SQLI/elevate-design-system.git
```

```css
/* In your main CSS file */
@import '@sqli/elevate-design-system/dist/sqli-design-system.css';
@import '@sqli/elevate-design-system/dist/fonts.css';
```

```html
<!-- Apply theme to your HTML -->
<html data-theme="sqli-light">
```

### Option 2: Platform.sh (No build required)

```html
<!DOCTYPE html>
<html data-theme="sqli-light">
<head>
  <link rel="stylesheet" href="https://main-bvxea6i-n3mp4dy5ud5ck.fr-3.platformsh.site/assets/sqli-design-system.min.css" />
  <link rel="stylesheet" href="https://main-bvxea6i-n3mp4dy5ud5ck.fr-3.platformsh.site/assets/fonts.css" />
</head>
<body>
  <button class="btn btn-primary">Hello World</button>
</body>
</html>
```

---

## Your First Component

Once installed, you can use any DaisyUI component with SQLI themes:

```html
<!-- Buttons -->
<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>

<!-- Card -->
<div class="card bg-base-100 shadow-xl">
  <div class="card-body">
    <h2 class="card-title">Hello World</h2>
    <p>Welcome to the Elevate Design System!</p>
  </div>
</div>

<!-- Alert -->
<div class="alert alert-info">
  <span>You're all set!</span>
</div>
```
