# Elevate Design System

**Version 0.5.0-alpha** - Complete design system for SQLI internal applications.

---

## Quick Start

```bash
npm install @sqli/elevate-design-system
```

```css
@import '@sqli/elevate-design-system';
```

```html
<html data-theme="sqli-light">
  <button class="btn btn-primary">Hello World</button>
</html>
```

## Brand Essentials

- **Logo**: "sqli" always in **lowercase**, write **SQLI** in text
- **Promise**: **"Elevate. Digitally"** (period only after "Elevate")
- **Tone**: Professional but human, no corporate jargon
- **Vocabulary**: Elevate, Transform, Inspire, Uplift, Raise, Amplify

See [Branding](?path=/docs/foundations-branding--docs) for logo variants and usage guidelines.

---

## Accessibility

- WCAG 2.1 AA compliant color contrast
- Full keyboard navigation support
- ARIA attributes on all components
- Visible focus states
- Reduced motion support
