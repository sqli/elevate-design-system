## Method 1: Git Repository (Recommended)

Install directly from GitLab Paris repository using npm:

### Prerequisites

- Node.js >= 22.0.0
- npm >= 9.0.0
- Access to SQLI GitLab Paris

### Installation

```bash
# Latest stable version (main branch)
npm install git+https://gitlab-paris.sqli.com/SQLI/elevate-design-system.git

# Specific version tag
npm install git+https://gitlab-paris.sqli.com/SQLI/elevate-design-system.git#v1.0.0

# Development branch (latest features)
npm install git+https://gitlab-paris.sqli.com/SQLI/elevate-design-system.git#dev
```

### Import CSS

```css
/* In your main CSS file */
@import '@sqli/elevate-design-system/dist/sqli-design-system.css';

/* Or minified version */
@import '@sqli/elevate-design-system/dist/sqli-design-system.min.css';

/* Include fonts (recommended) */
@import '@sqli/elevate-design-system/dist/fonts.css';
```

```javascript
// In your JS entry point
import '@sqli/elevate-design-system/dist/sqli-design-system.css';
import '@sqli/elevate-design-system/dist/fonts.css';
```

### Apply Theme

```html
<html data-theme="sqli-light">
  <!-- Your content -->
</html>
```

## Advanced: Tailwind Preset

Extend your Tailwind configuration with SQLI colors:

```javascript
// tailwind.config.js
import sqliPreset from '@sqli/elevate-design-system/preset';

export default {
  presets: [sqliPreset],
  // ... your config
}
```

This adds:

- `sqli-cream`, `sqli-midnight`, `sqli-cobalt`, `sqli-sky` colors
- TWK Everett as default sans-serif font

## Theme Switching

### JavaScript Toggle

```javascript
function toggleTheme() {
  const html = document.documentElement;
  const current = html.getAttribute('data-theme');
  const next = current === 'sqli-light' ? 'sqli-dark' : 'sqli-light';
  html.setAttribute('data-theme', next);
  localStorage.setItem('theme', next);
}

// Load saved theme
const saved = localStorage.getItem('theme') || 'sqli-light';
document.documentElement.setAttribute('data-theme', saved);
```

### System Preference

```javascript
function getSystemTheme() {
  return window.matchMedia('(prefers-color-scheme: dark)').matches
    ? 'sqli-dark'
    : 'sqli-light';
}

// Apply system theme if no preference saved
const theme = localStorage.getItem('theme') || getSystemTheme();
document.documentElement.setAttribute('data-theme', theme);
```

### DaisyUI Theme Controller

```html
<input type="checkbox" value="sqli-dark" class="toggle theme-controller" />
```

## Font Integration

### Option 1: Automatic (Default)

Fonts are loaded automatically via CSS. Ensure font files are accessible at `/fonts/` or configure your bundler.

### Option 2: Platform.sh Fonts

```html
<link rel="stylesheet" href="https://main-bvxea6i-n3mp4dy5ud5ck.fr-3.platformsh.site/assets/fonts.css" />
```

### Option 3: Self-Hosted

```bash
# Copy fonts to your project
cp -r node_modules/@sqli/elevate-design-system/dist/fonts ./public/fonts
```

### Font Preloading (Performance)

```html
<link rel="preload"
      href="/fonts/TWKEverett-Regular.woff2"
      as="font"
      type="font/woff2"
      crossorigin />
```
