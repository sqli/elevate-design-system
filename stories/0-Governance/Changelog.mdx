## [0.12.0-alpha] - 2026-02-16

Major architecture upgrade from CSS-only DaisyUI to a full React component library.

### React Component Library

- **25 typed React components** with CVA (class-variance-authority) variants
- **forwardRef** pattern on all components for ref forwarding
- **cn() utility** (clsx + tailwind-merge) for class merging

### Radix UI Integration

- **12 accessible primitives**: Dialog, Accordion, Tabs, Select, Checkbox, RadioGroup, Switch, Slider, DropdownMenu, Tooltip, Toast, Label
- Full keyboard navigation and ARIA attributes out of the box

### ElevateTheme Provider

- `<ElevateTheme>` wrapper with `useTheme()` hook
- `useThemeOptional()` for components outside the provider
- Appearance management: light, dark, system
- OS preference detection via `prefers-color-scheme`

### Semantic CSS Tokens

- **Shadow scale**: `--shadow-1` through `--shadow-6`
- **Overlay**: `--color-overlay`
- **Surface & Panel**: `--color-surface`, `--color-panel-solid`, `--color-panel-translucent`
- **Focus**: `--color-focus`

### TypeScript

- Full type safety with `.d.ts` exports
- VariantProps from CVA for type-safe variant props
- Strict mode enabled

### Build System

- **tsup** for component builds (ESM + CJS + `.d.ts`)
- Dual package exports (`import` / `require`)
- Tree-shakeable ESM output

### Recharts Integration

- Chart theming with SQLI brand colors
- Bar, Line, Area, Pie, and Radar chart variants

## [0.5.0-alpha] - 2026-01-05

Initial public release of the Elevate Design System.

### Features

- **DaisyUI Integration** - 65+ CSS components with SQLI themes
- **SQLI Themes** - `sqli-light` (Cream/Cobalt) and `sqli-dark` (Midnight/Sky)
- **TWK Everett Font** - Custom SQLI typeface (Light, Regular, Medium, Bold)
- **Brand Assets** - Logos, Ascenders, favicons in multiple formats
- **Design Tokens** - CSS variables for colors, typography, spacing
- **Tailwind CSS 4** - Modern CSS framework integration
- **WCAG 2.1 AA** - Accessible color contrast and focus states

### Distribution

- **npm Package** - `@sqli/elevate-design-system`
- **CDN Support** - unpkg and jsDelivr
- **Multiple Exports** - Full CSS, minified, tokens-only, fonts, Tailwind preset

### Documentation

- **Storybook** - Interactive component documentation
- **Getting Started** - 5-minute quick start guide
- **Installation Guide** - npm, CDN, framework examples
- **Design Tokens** - CSS variables reference
- **Theming Guide** - Light/dark mode switching
