## Method 1: npm (Recommended)

### Prerequisites

- Node.js >= 22.0.0
- npm >= 9.0.0
- Access to SQLI GitLab registry

### Step 1: Configure npm Registry

Create or modify `.npmrc` at your project root:

```ini
# @sqli scope configuration
@sqli:registry=https://gitlab.paris.sqli.com/api/v4/projects/PROJECT_ID/packages/npm/

# Authentication
//gitlab.paris.sqli.com/api/v4/projects/PROJECT_ID/packages/npm/:_authToken=${GITLAB_TOKEN}
```

### Step 2: Set GitLab Token

```bash
# Linux/macOS
export GITLAB_TOKEN="your-gitlab-token"

# Windows PowerShell
$env:GITLAB_TOKEN="your-gitlab-token"
```

### Step 3: Install

```bash
npm install @sqli/elevate-design-system
```

### Step 4: Import CSS

```css
/* In your main CSS file */
@import '@sqli/elevate-design-system';
```

### Step 5: Apply Theme

```html
<html data-theme="sqli-light">
  <!-- Your content -->
</html>
```

## Method 2: CDN (unpkg)

No build step required. Add directly to your HTML:

```html
<!DOCTYPE html>
<html lang="en" data-theme="sqli-light">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My App</title>

  <!-- Elevate Design System CSS -->
  <link rel="stylesheet"
        href="https://unpkg.com/@sqli/elevate-design-system@0.5.0-alpha/dist/sqli-design-system.min.css" />

  <!-- Fonts (required for TWK Everett) -->
  <link rel="stylesheet"
        href="https://unpkg.com/@sqli/elevate-design-system@0.5.0-alpha/dist/fonts.css" />
</head>
<body class="bg-base-100 text-base-content">
  <button class="btn btn-primary">Hello World</button>
</body>
</html>
```

## Method 3: CDN (jsDelivr)

jsDelivr offers better global CDN coverage and caching:

```html
<!-- Fonts -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/@sqli/elevate-design-system@0.5.0-alpha/dist/fonts.css" />
```

## Advanced: Tokens Only

Import only the CSS variables without DaisyUI components:

```css
/* npm */
@import '@sqli/elevate-design-system/tokens';

/* CDN */
<link rel="stylesheet" href="https://unpkg.com/@sqli/elevate-design-system/dist/tokens.css" />
```

Available variables:

```css
:root {
  --color-sqli-cream: #FFFAF0;
  --color-sqli-midnight: #0F0E2B;
  --color-sqli-cobalt: #1F24E9;
  --color-sqli-sky: #6DA5FF;
  --font-family-sans: 'TWK Everett', system-ui, sans-serif;
}
```

## Advanced: Tailwind Preset

Extend your Tailwind configuration with SQLI colors:

```javascript
// tailwind.config.js
import sqliPreset from '@sqli/elevate-design-system/preset';

export default {
  presets: [sqliPreset],
  // ... your config
}
```

This adds:

- `sqli-cream`, `sqli-midnight`, `sqli-cobalt`, `sqli-sky` colors
- TWK Everett as default sans-serif font

## Framework Examples

### Next.js (App Router)

```css
/* app/globals.css */
@import '@sqli/elevate-design-system';
```

```tsx
// app/layout.tsx

  return (
    <html lang="en" data-theme="sqli-light" suppressHydrationWarning>
      <body>{children}</body>
    </html>
  );
}
```

### Vite + React

```css
/* src/index.css */
@import '@sqli/elevate-design-system';
```

```tsx
// src/main.tsx
import './index.css';
import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';

document.documentElement.setAttribute('data-theme', 'sqli-light');
```

## Theme Switching

### JavaScript Toggle

```javascript
function toggleTheme() {
  const html = document.documentElement;
  const current = html.getAttribute('data-theme');
  const next = current === 'sqli-light' ? 'sqli-dark' : 'sqli-light';
  html.setAttribute('data-theme', next);
  localStorage.setItem('theme', next);
}

// Load saved theme
const saved = localStorage.getItem('theme') || 'sqli-light';
document.documentElement.setAttribute('data-theme', saved);
```

### System Preference

```javascript
function getSystemTheme() {
  return window.matchMedia('(prefers-color-scheme: dark)').matches
    ? 'sqli-dark'
    : 'sqli-light';
}

// Apply system theme if no preference saved
const theme = localStorage.getItem('theme') || getSystemTheme();
document.documentElement.setAttribute('data-theme', theme);
```

### DaisyUI Theme Controller

```html
<input type="checkbox" value="sqli-dark" class="toggle theme-controller" />
```

## Font Integration

### Option 1: Automatic (Default)

Fonts are loaded automatically via CSS. Ensure font files are accessible at `/fonts/` or configure your bundler.

### Option 2: CDN Fonts

```html
<link rel="stylesheet" href="https://unpkg.com/@sqli/elevate-design-system/dist/fonts.css" />
```

### Option 3: Self-Hosted

```bash
# Copy fonts to your project
cp -r node_modules/@sqli/elevate-design-system/dist/fonts ./public/fonts
```

### Font Preloading (Performance)

```html
<link rel="preload"
      href="/fonts/TWKEverett-Regular.woff2"
      as="font"
      type="font/woff2"
      crossorigin />
```
